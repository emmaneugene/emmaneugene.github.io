# Personal website

Source files and assets for my personal site, written in plain HTML/CSS/JS with some helper libraries.

## Preview

![Preview](./assets/preview.png)

## Local development

```bash
# Serve with live-server (defaults to port 8080)
npm run dev

# Format with prettier
npm run format
```
